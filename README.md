# ClubOps – Club Administration System

ClubOps is a role-based club management platform developed for managing college club activities, event proposals, and budget approvals. The system includes separate dashboards for admins and club heads with responsive UI and workflow management features.

## Features

* Role-based login system
* Admin and Club Head dashboards
* Event proposal management
* Budget approval workflow
* Submission tracking
* Responsive modern UI
* Local storage-based data handling

## Tech Stack

* HTML
* CSS
* JavaScript

## Modules

### Admin Features

* Event approval/rejection
* Budget approval system
* Dashboard statistics
* Event registry management

### Club Head Features

* Propose events
* Submit budget requests
* Track submissions
* Profile management
